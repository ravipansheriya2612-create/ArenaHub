# ArenaHub

A modern MERN Stack Sports Ground Booking Platform.

## Features

- JWT Authentication
- Role Based Admin
- Sports Ground Booking
- Razorpay Payment Gateway
- Cloudinary Image Upload
- AI Review Enhancement (Gemini)
- Reviews & Ratings
- Responsive UI
- Booking Management
- Admin Dashboard

## Tech Stack

Frontend
- React
- Tailwind CSS
- Axios
- React Router
- React Hot Toast
- Lucide React

Backend
- Node.js
- Express.js
- MongoDB
- JWT
- Multer
- Cloudinary
- Razorpay
- Gemini API

## Installation

Backend

```bash
cd Backend
npm install
npm run dev
```

Frontend

```bash
cd Frontend
npm install
npm run dev
```
